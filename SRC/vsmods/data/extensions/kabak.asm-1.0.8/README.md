## Features ##
-----------------------------------------------------------------------------------------------------------
Pic18 Assembler syntax highlighter

* Fully supported MPLABX 5.30 mpasmx syntax

* Fully supported names for pic18f26k22 registers 
* Fully supported names for pic18f04-15q41 registers



## Ussage 
-----------------------------------------------------------------------------------------------------------
Best viewed with default dark theme
